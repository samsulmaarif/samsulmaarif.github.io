Shell di Linux

Apa itu shell?

Shell adalah program yang digunakan untuk menghubungkan user dengan kernel sistem operasi, dalam hal ini Linux kernel. Setiap kali perintah yang dimasukkan oleh user akan diterjemahkan oleh shell kemudian hasilnya akan dikirimkan ke kernel dan kernel inilah yang akan melakukan operasi yang diminta oleh user tersebut.

Shell di Linux (2)

Macam-macam shell di Linux antara lain:

  • Bourne Shell (sh).
  • C Shell (csh).
  • Korn Shell (ksh).
  • Bourne Again Shell (bash).
  • Almquist shell (ash).
  • BusyBox.
  • Debian Almquist shell (dash).
  • Exensible shell (es).
  • Friendly interactive shell (fish).
  • Stand Alone Shell (sash).
  • Z shell (zsh).
  • Hart to Use shell (hush).
  • Karel’s Interactive Simple shell/Karel’s Incredibly Stupid shell (kiss).

Perintah Dasar Linux

Sebuah shell pada Linux pada umumnya memiliki format :

[nama_pengguna]@[hostname]:~$

Perintah Dasar Linux (2)

Prompt shell di Linux ada 2, $ dan #

  • $ : adalah prompt untuk user biasa.

    Contohnya

    samsul@blankon:~$
  • # : adalah prompt untuk pengguna dengan hak akses root.

    Contohnya

    root@blankon:~#

Perintah Dasar Linux (3)

Format perintah Linux standar mempunyai format sebagai berikut:

$ nama_perintah [pilihan] [argumen]

Contoh:

$ ls tanpa argumen
$ ls -a option adalah -a=all, tanpa argumen
$ ls /bin tanpa option, argumen adalah /bin
$ ls /bin /etc /usr ada 3 argumen
$ ls -l /usr 1 option dan 1 argumen
$ ls -la /bin /etc 2 option -l dan -a dan 2 argumen

Perintah Dasar Linux (7)

Baca manual/panduan/dokumentasi perintah!!!

samsul@blankon:~$ man nama_perintah

dengan opsi --help atau -h

samsul@blankon:~$ nama_perintah --help

dengan perintah info

samsul@blankon:~$ info nama_perintah

Perintah sudo dan su

  • sudo
    merupakan perintah yang dapat digunakan untuk menjalan perintah dengan hak akses "root" (super user) oleh pengguna biasa. Seorang administrator dapat mendelegasikan tugas administratifnya kepada pengguna lain tanpa memberikan password root kepada pengguna tersebut.
  • su
    hampir sama dengan sudo, perbedaannya adalah saat menjalankan perintah ini password yang digunakan adalah password root, bukan password pengguna biasa. Jadi administrator harus memberikan password root kepada pengguna yang akan menjalankan perintah ini.

Perintah sudo dan su (2)

Contoh penggunaannya :

  • sudo
    samsul@blankon:~$ sudo poweroff
[sudo] password for samsul:
  • su
    samsul@blankon:~$ su -c poweroff
Password:

* Secara default Ubuntu tidak mengaktifkan password root.
* Password di Linux tidak terlihat, ketikkan saja passwordnya
   lalu tekan ENTER.

Tips Keren

Cara efektif bekerja dengan Bash

  • Gunakan fitur auto complete, ketikkan awal perintah (atau argumen) lalu tekan [TAB] di keyboard 

    samsul@blankon:~$ ls /ho [TAB]

Tips Keren (2)

  • Gunakan pintasan keyboard (shortcut)

    Shortcut Navigasi Deskripsi
    Arah Kiri/kanan Berpindah ke kiri/kanan
    Ctrl+A Berpindah ke awal baris
    Ctrl+E Berpindah ke akhir baris
    Ctrl+Arah Kanan Berpindah ke kanan satu kata
    Ctrl+Arah Kiri Berpindah satu kata ke kiri

Tips Keren (3)

  • Gunakan pintasan keyboard (shortcut)

    Shortcut Editing Deskripsi
    Ctrl+U Hapus teks/perintah dari posisi kursor ke awal baris
    Ctrl+K Hapus teks/perintah dari posisi kursor ke akhir baris
    Ctrl+W Hapus kata/perintah di belakang kursor
    Ctrl+D Hapus karakter dari depan (delete)
    Ctrl+T Pindahkan karakter dari kiri dan kanan kursor
    Alt+T Pindahkan kata dari kiri dan kanan kursor

  • Gunakan pintasan keyboard (shortcut)
